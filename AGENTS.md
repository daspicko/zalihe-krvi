# Project instructions for using AI Agents

This document provides guidelines and instructions for using AI agents effectively in your projects. AI agents can automate tasks, provide insights, and enhance user experiences. Follow the steps below to integrate AI agents into your workflow.

## Code style
- **HTML5**: Structure your web content using HTML5 for semantic and accessible markup.
- **CSS3**: Utilize CSS3 for styling and layout to ensure responsive and visually appealing designs.
- **JavaScript (ES6+)**: Use modern JavaScript features and syntax for better readability and maintainability. Lint your code with ESLint.

## Architecture
- All JavaScript code is located in the `js/` directory.
- All CSS files are stored in the `css/` directory.
- All images are kept in the `assets/images/` directory.
- All fonts are stored in the `assets/fonts/` directory.
- There is web scraping functionality in the `scraper/` directory. It contains scripts to gather data from various sources.

## Ways of working
- Always work on a single taks at a time to maintain focus and efficiency.
- Every prompt should be executed step-by-step and get feedback before proceeding to the next step.
- After making changes, run tests to ensure nothing is broken and then display a summary and provide me a review of the modifications. I always want to review the changes before they are finalized. Ensure cycle your changes -> tests -> summary -> review.

## Permissions
- DO NOT edit or delete any files outside your designated project folder without explicit permission.
- DO NOT commit or edit git history unless user explicitly instructs you to do so.
- DO NOT run any teminal commands without explicit permmission.
